# 自动战斗辅助

0.简介
1.策略组设置

## 简介

自动战斗辅助可以按照设定的角色、策略、优先级、触发条件等自动切换角色和执行攻击，使用元素战技和元素爆发。

## 策略组

自动战斗辅助支持以下策略：
| 策略 | 说明 |
| ----------------- | --------------------------------------------------------- |
| `>`               | 立刻跳转至下一组    |
| `@e?`        | 元素战技是否生效 |
| `e?`      | 元素战技是否就绪            |
| `a`/`a~`          | 普通攻击/重击             |
| `e`/`e~`          | 使用元素战技（点按/长按）           |
| `j`               | 跳跃                                              |
| `ja`              | 跳跃攻击                                 |
| `sp`               | 冲刺                                            |
| `r`               | 瞄准                                               |
| `rr`              | 快速瞄准两次                           |
| 数字   | 延时，单位为`毫秒`    |
